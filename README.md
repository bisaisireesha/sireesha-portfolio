# Bisai Sireesha - Portfolio

A premium, interactive portfolio website built with HTML, CSS, and Vanilla JavaScript. Features 3D animations, glassmorphism, and a celestial theme.

## 🚀 Features
- **3D Interactive Elements**: Rotating skill cube and animated background rings.
- **Modern UI/UX**: Glassmorphism, smooth scroll reveals, and a custom interactive cursor.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Performance**: Lightweight and fast-loading with no external frameworks.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom properties, Keyframe animations)
- Vanilla JavaScript (Intersection Observer, custom DOM interactions)
- Google Fonts (Space Grotesk & Syne)

## 📦 How to Run
1. Clone the repository
2. Open `index.html` in any modern browser.
